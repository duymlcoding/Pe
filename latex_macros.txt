CUSTOM TCOLORBOX ENVIRONMENTS:
------------------------------
\newtcolorbox{conceptbox}[1][]
  - Purpose: Key concepts and principles
  - Converted to: {note} admonition in Markdown
  - Optional title parameter

\newtcolorbox{formulabox}[1][]
  - Purpose: Important formulas and equations
  - Converted to: {important} admonition in Markdown
  - Optional title parameter

\newtcolorbox{examplebox}[2][]
  - Purpose: Example problems
  - Converted to: {prf:example} in Markdown
  - Required title parameter (#2)

\newtcolorbox{stepbox}[1][]
  - Purpose: Solution steps
  - Converted to: {dropdown} "Solution Steps" in Markdown

\newtcolorbox{keybox}[1][]
  - Purpose: Key variables and definitions
  - Converted to: {admonition} with :class: tip in Markdown
  - Optional title parameter

FORMATTING COMMANDS:
-------------------
\renewcommand{\baselinestretch}{1.1}

These were removed in Markdown conversion as spacing is handled by CSS.

SECTION FORMATTING:
------------------
\titleformat{\section}
  {\normalfont\LARGE\bfseries\color{conceptcolor}}
  {\thesection}{1em}{}

\titleformat{\subsection}
  {\normalfont\Large\bfseries\color{conceptcolor}}
  {\thesubsection}{1em}{}

Converted to standard Markdown headings (## and ###).
